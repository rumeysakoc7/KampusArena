# KampusArena
KampusArena, üniversite kampüsü içerisinde öğrencilerin kendi aralarında ürün alım–satım ve kiralama işlemlerini gerçekleştirebilmeleri amacıyla geliştirilmiş bir masaüstü uygulamasıdır. Proje C# Windows Forms kullanılarak geliştirilmiş olup veritabanı işlemleri SQL Server ile gerçekleştirilmiştir. Kullanıcılar sisteme kayıt olup giriş yapabilir, ürün ekleyebilir, ürünlerini satışa veya kiralamaya sunabilir ve diğer kullanıcıların ürünlerini inceleyerek satın alma veya kiralama talebi oluşturabilir. Kiralama işlemlerinde kullanıcılar belirli tarih aralıkları seçerek işlem yapabilir ve sistem daha önce kiralanmış tarih aralıklarının tekrar seçilmesini engelleyerek çakışmaları önler. Ayrıca kullanıcılar sistem içinden mesajlaşabilir, ürünler için teklif verebilir ve kiralama süresi yaklaşan ürünler için otomatik e-posta bildirimi alabilir. Satış ve kiralama işlemlerinde ödeme süreci admin onayı ile tamamlanmaktadır.

## Özellikler
-Kullanıcı kayıt ve giriş sistemi
-Ürün ekleme, düzenleme ve silme
-Satış ve kiralama işlemleri
-Tarih aralığına göre kiralama kontrolü
-Kategori sistemi
-Ürün teklif verme
-Sistem içi mesajlaşma
-Bildirim sistemi
-Admin kontrol paneli
-Ödeme onay sistemi
-Kiralama süresi yaklaşınca otomatik e-posta gönderimi

## Kullanılan Teknolojiler
-C#
-.NET Windows Forms
-SQL Server
-Microsoft.Data.SqlClient
-SMTP (E-posta gönderimi)
-Git & GitHub
-Kurulum

## Projeyi çalıştırmak için:
-Projeyi indir veya klonla
-SQL Server’da KampusArena veritabanını oluştur
-DatabaseHelper.cs dosyasındaki connection string ayarını kendi SQL Server adınıza göre düzenleyin
-Visual Studio ile projeyi açıp çalıştırın

## Amaç
Bu proje, C# ve SQL Server teknolojileri kullanılarak masaüstü uygulama geliştirme pratiği kazanmak amacıyla geliştirilmiştir.
